# CV

This repository contains the typst code for my cv.
You can build it by executing `pixi run compile` or `pixi run watch` in the root directory.
The CV is using the [moderner-cv](https://github.com/pavelzw/moderner-cv) template.
Check out the latest version [here](https://github.com/pavelzw/cv/releases/latest/download/cv.pdf).
