# CV

[![CV](https://img.shields.io/badge/CV-latest-ec4899?style=flat-square)](https://pavel.pink/cv)

This repository contains the typst code for my cv.
You can build it by executing `pixi run compile` or `pixi run watch` in the root directory.
The CV is using the [moderner-cv](https://github.com/pavelzw/moderner-cv) template.
Check out the latest version [here](https://pavel.pink/cv).

Use [skim](https://skim-app.sourceforge.io/) to view the PDF on macOS while editing.
